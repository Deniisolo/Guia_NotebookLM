# NotebookLM — Guía educativa

Sitio web estático en español que presenta [Google NotebookLM](https://notebooklm.google.com): qué es, cómo usarlo, tutoriales recomendados y un taller práctico para estudiantes.

## Contenido

- **¿Qué es?** — Introducción a NotebookLM y tipos de fuentes que acepta
- **Funciones** — Resúmenes, chat con citas, Audio Overview, mapas mentales y cuestionarios
- **Cómo usar** — Guía paso a paso desde cero
- **Tutoriales** — Enlaces a videos en YouTube (español e inglés)
- **Taller** — Actividades prácticas con entregables (configuración, fuentes, prompts y podcast)

## Estructura del proyecto

```
NotebookLM/
├── Index.html   # Estructura y contenido de la página
├── styles.css   # Estilos (variables, layout, animaciones, responsive)
└── README.md
```

No hay dependencias ni proceso de compilación: solo HTML, CSS y un pequeño script para el acordeón del taller.

## Requisitos para el taller

- Computador con internet
- Cuenta de Google (Gmail)
- Acceso a [notebooklm.google.com](https://notebooklm.google.com)

## Tecnologías

- HTML5
- CSS3 (variables custom, grid, flexbox, media queries)
- JavaScript vanilla (acordeón de actividades)
- Fuentes: [Syne](https://fonts.google.com/specimen/Syne) y [DM Sans](https://fonts.google.com/specimen/DM+Sans) (Google Fonts)
